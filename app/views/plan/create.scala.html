Define an action plan

Each plan consists of a set of rules or instructions for changes in hub content.

Beyond a name and description, the 'fulfill' rule governs how items that you are currently subscribed to ought to be handled: 'deliver' means transfer the packaged content using the destination supplied (usually SWORD deposit), 'review' means placing the item on the hold list for your review, and 'notify' would cause an message to be sent.
