Content Formats

The hub is completely ignorant about how the content put in it is formatted, so the analyst must specify this information. You may install as many formats as needed, and then you may begin to add tools expecting these formats. For example, scholarly published articles in many disciplines are routinely produced with a standard XML schema (e.g. NLM-DTD/JATs), and this format is widely understood.
